# Registration Form Validation

This project is a simple registration form created using HTML, CSS, and JavaScript. The main purpose of this task is to practice real-time form validation.

In this project, I created a form with Name, Email, and Password fields and added validation using JavaScript.

## Features
- Name field cannot be empty  
- Email must be in a valid format  
- Password must be at least 6 characters  
- Error messages are displayed for invalid inputs  
- Submit button is disabled until all inputs are valid  
- Validation happens while the user types  

## Technologies Used
- HTML  
- CSS  
- JavaScript
